Nasze projekty

Aby dowiedzieć się więcej kliknij na wybrany projekt

Relive7

Nasza pierwsza modyfikacja. Bazuje ona na systemie Windows 10 2021 LTSC. Modyfikacja zmienia wygląd systemu praktycznie 1:1 w wygląd Windows 7.

ReliveVista

ReliveVista to modyfikacja Windowsa 10 21H2 (dokładniej wersji IoT Enterprise LTSC), która próbuje jak najbardziej przywrócić wygląd z Windowsa Visty.

ReliveXP

Już wkrótce

Relive98

To jest modyfikacja Windowsa 10 LTSC 2021 z motywem Windowsa 98 Second Edition. Modyfikacja będzie przywracać UI z systemu Windows 98, i przerobione programy.

OldBuntu

To modyfikacja wychodząca z naszego projektu. Bazuje ona na Ubuntu 21.04, i przywraca klasyczny interfejs systemu, programy oraz wiele więcej. Cały mod inspierowany jest wyglądem Ubuntu 15.04. Modyfikacja ta powstała, specjalnie dla fanatyków starego wyglądu Ubuntu, i osób, którym nie podoba się obecne środowisko graficzne czyli "Gnome".

Project CEMeNT

Ten projekt nie jest mocno modyfikacją interfejsu graficznego systemu Windows. Jest to mod w stylu Atlas OS, upodobniony do różnych wersji systemu Windows CE z podobnymi okrojeniami z CE.

Relive Marketplace

Relive Marketplace to sklep z przydatnymi programami, lub projektami które polecają sami twórcy projektów Relive! Znajdziesz tam np. Aplikacje komunikacyjne, czy motywy do przeglądarek.

Revert8Plus

Revert8Plus to paczka która zamienia wygląd twojego aktualnego systemu Windows 8.x - Windows 11 na wygląd systemów Windows Vista - 7 (wkrótce Windows XP - 8.x)

Reunion7

Reunion7 to modyfikacja Windows 10 IoT Enterprise LTSC 2021 zamieniająca system w Windows 7 (tak samo jak Relive7), tylko że po angielsku.

Inne narzędzia:

Geckium - Motyw CSS do przeglądarki Mozilla Firefox 115 ESR przywracający wygląd znany użytkownikowi ze starych wersji Google Chrome.
Echelon - Motyw CSS do przeglądarki Mozilla Firefox 115 ESR przywracający wygląd znany użytkownikowi ze starych wersji przeglądarki Mozilla Firefox.
Stare ikony - Kolekcja ikon do customizacji systemu.
